# 📊 카드 지출 관리 시스템 v2.0 - 업데이트 가이드

## 🎉 새로운 기능

### 1. 고급 분석 대시보드 📈

**Chart.js 기반 인터랙티브 차트**
- **월별 지출 추이**: 선 그래프로 지출 패턴 시각화
- **카테고리별 막대 차트**: Top 8 카테고리 한눈에 비교
- **지출 비율 도넛 차트**: 카테고리별 비율을 시각적으로 표현
- **상세 통계 테이블**: 카테고리별 총액, 건수, 비율, 평균 금액

**사용 방법:**
1. "고급분석" 탭 클릭
2. 기간 선택 (3개월/6개월/12개월)
3. 차트에서 데이터 포인트에 마우스를 올려 상세 정보 확인

### 2. 예산 관리 시스템 💰

**카테고리별 월 예산 설정**
- 카테고리별로 개별 예산 설정
- 실시간 사용률 추적
- 80% 도달 시 경고 알림
- 100% 초과 시 초과 금액 표시

**예산 상태 표시:**
- 🟢 **안전**: 80% 미만 사용
- 🟡 **경고**: 80-100% 사용
- 🔴 **초과**: 100% 이상 사용

**사용 방법:**
1. "예산관리" 탭 클릭
2. "+ 예산 추가" 버튼 클릭
3. 카테고리 선택 및 월 예산 입력
4. 자동으로 진행률 계산 및 알림

**예산 예시:**
- 식료품: 500,000원
- 외식: 300,000원
- 교통: 200,000원
- 문화/여가: 150,000원

### 3. Excel 리포트 자동 생성 📄

**5개 시트로 구성된 전문 리포트**

**시트 1 - 요약**
- 기간별 지출 총계
- 거래 건수
- 일평균 지출

**시트 2 - 월별 상세**
- 월별 카테고리 분석
- 전월 대비 증감

**시트 3 - 카테고리 분석**
- 카테고리별 통계
- 파이 차트 포함

**시트 4 - 거래 내역**
- 최근 1000건 거래 목록
- 날짜별 정렬

**시트 5 - 재무 목표**
- 목표별 진행 현황
- 달성률 계산

**다운로드 방법:**
1. "고급분석" 탭에서 "Excel 리포트 다운로드" 버튼 클릭
2. 자동으로 날짜가 포함된 파일명으로 저장

### 4. CSV 내보내기 📋

**거래 내역을 CSV 형식으로 내보내기**
- 다른 프로그램에서 활용 가능
- 날짜 범위 지정 가능
- 최대 10,000건까지 내보내기

**API 사용:**
```bash
GET http://localhost:8000/api/export/csv?start_date=2024-01-01&end_date=2024-12-31
```

- 예산 관리 테이블 추가
- 인덱스 최적화로 조회 속도 향상

### 사용자 인터페이스
- 차트 반응형 디자인
- 터치 최적화 (모바일/태블릿)
- 로딩 상태 표시 개선

### API 엔드포인트

**예산 관리**
```
POST   /api/budgets          - 예산 생성
GET    /api/budgets          - 예산 조회
PUT    /api/budgets/{id}     - 예산 수정
DELETE /api/budgets/{id}     - 예산 삭제
```

**데이터 내보내기**
```
POST   /api/export/excel     - Excel 리포트 생성
GET    /api/export/csv       - CSV 내보내기
```

## 📖 사용 시나리오

### 시나리오 1: 월 초 예산 계획
```
1. "예산관리" 탭에서 이번 달 예산 설정
2. 각 카테고리별로 목표 금액 입력
3. 한 달 동안 실시간으로 사용률 추적
4. 80% 도달 시 지출 조절
```

### 시나리오 2: 월말 지출 분석
```
1. "고급분석" 탭에서 이번 달 패턴 확인
2. 전월 대비 증가한 카테고리 파악
3. Excel 리포트 다운로드하여 배우자와 공유
4. 다음 달 예산 재조정
```

### 시나리오 3: 재무 목표 점검
```
1. "재무목표" 탭에서 주택구입/노후자금 진행률 확인
2. 목표 달성을 위한 월 저축액 재계산
3. 불필요한 지출 카테고리 파악
4. 예산 조정 및 지출 최적화
```

## 💡 활용 팁

### Tip 1: 예산 알림 활용
- 각 카테고리별로 실제 월 평균 지출의 110%를 예산으로 설정
- 80% 경고가 뜨면 남은 기간 계산하여 지출 조절

### Tip 2: Excel 리포트 활용
- 월말마다 리포트 다운로드하여 보관
- 연말에 12개월 리포트 비교 분석
- 배우자/가족과 공유하여 재무 상황 투명하게 관리

### Tip 3: 카테고리 최적화
- 처음 3개월은 기본 카테고리로 데이터 수집
- 지출 패턴 파악 후 세부 카테고리 추가
- 불필요한 카테고리는 "기타"로 통합

### Tip 4: 데이터 백업
```bash
# 데이터베이스 백업
cp data/expenses.db data/backup_$(date +%Y%m%d).db

# 정기적 백업 (매월 1일)
0 0 1 * * cp /path/to/data/expenses.db /path/to/backup/expenses_$(date +\%Y\%m).db
```

## 🔧 문제 해결

### 차트가 표시되지 않는 경우
```bash
# Chart.js 재설치
cd frontend
npm install chart.js@latest
npm run dev
```

### Excel 다운로드 실패
```bash
# openpyxl 재설치
cd backend
pip install openpyxl --upgrade --break-system-packages
```

### 예산이 저장되지 않는 경우
```bash
# 데이터베이스 테이블 확인
sqlite3 data/expenses.db "SELECT * FROM category_budgets;"

# 필요시 테이블 재생성
python -c "from database import Database; db = Database(); db.init_db()"
```
